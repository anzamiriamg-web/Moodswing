### Project Description
Post a short mood message or choose from mood tags like happy, anxious, calm, excited, lonely, overwhelmed, etc.
If someone‚Äôs mood resonates with you, send them a thoughtful message or emoji reaction. Positive, supportive interactions are encouraged.

### The Problem statement
Allows users to easily express their current mood. 
Encourages supportive and meaningful conversations.

### The Solution
Users can:
Select or type their current mood
Use mood tags (e.g., happy, stressed, lonely, excited).
This allows easy emotional expression without pressure.
A real-time feed displays moods shared by other users.
Send supportive messages or emojis.
